برنامج إدارة المؤتمر - نسخة أونلاين بقاعدة بيانات مشتركة

هذه النسخة تجعل كل المستخدمين يرون نفس البيانات في نفس الوقت، بشرط تشغيلها على سيرفر واحد أونلاين.

طريقة التشغيل على الكمبيوتر للتجربة:
1) فك الضغط عن الملف.
2) افتح Terminal داخل مجلد البرنامج.
3) نفذ:
   npm install
   npm start
4) افتح:
   http://localhost:3000

طريقة الاستخدام كموقع/تطبيق موبايل:
- ارفع هذا المجلد على استضافة تدعم Node.js و WebSocket.
- لازم يكون فيها قرص دائم/Persistent Disk حتى لا تضيع قاعدة البيانات عند إعادة تشغيل السيرفر.
- قاعدة البيانات ستكون في:
  data/conference.db
- بعد رفع الموقع، افتح الرابط من الموبايل ثم اختر Add to Home screen.

مهم جدًا:
- لو فتحت ملف index.html مباشرة من غير السيرفر، سيعمل كنسخة محلية فقط ولن تتزامن البيانات.
- لو فتحت الرابط المنشور على السيرفر، ستعمل المزامنة بين كل الأجهزة.
- أول جهاز يفتح التطبيق على السيرفر الفارغ يرفع بياناته الحالية إلى قاعدة البيانات.

تشغيل سريع على شبكة داخلية:
- شغل npm start على جهاز كمبيوتر.
- افتح من الموبايل: http://IP-ADDRESS:3000
- هذه الطريقة تعمل داخل نفس شبكة Wi-Fi فقط.
